# Pathfinder

Приложение для поиска кратчайшего пути на клеточном поле с визуализацией работы алгоритмов.

## Возможности программы

- создание и редактирование карты;
- установка точки старта;
- установка точки финиша;
- добавление и удаление препятствий;
- поиск пути алгоритмом **BFS**;
- поиск пути алгоритмом **DFS**;
- поиск пути алгоритмом **A\***;
- пошаговая визуализация поиска;
- отображение посещённых вершин;
- отображение очереди, стека и приоритетной очереди;
- сравнение длины найденного пути;
- импорт карты из файла;
- экспорт результатов поиска в текстовый файл;
- отображение статистики работы алгоритмов.

---

## Используемые алгоритмы

### BFS (Поиск в ширину)

Использует очередь (Queue). Гарантированно находит кратчайший путь при одинаковой стоимости переходов.

### DFS (Поиск в глубину)

Использует стек (Stack). Выполняет поиск в глубину и не гарантирует нахождение кратчайшего пути.

### A*

Использует приоритетную очередь и эвристику Манхэттенского расстояния. Обычно находит кратчайший путь быстрее, чем BFS.

## Запуск программы

Убедитесь, что установлен Python 3.10 или выше.

Запуск:

```bash
python main.py
```

---

## Работа с программой

1. Нарисуйте препятствия.
2. Установите стартовую точку.
3. Установите конечную точку.
4. Выберите алгоритм поиска (BFS, DFS или A*).
5. Дождитесь завершения поиска.
6. При необходимости сохраните карту или экспортируйте результат.
