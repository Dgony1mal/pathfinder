# Pathfinder

Приложение для поиска кратчайшего пути на клеточном поле с визуализацией работы алгоритмов.

## Возможности программы

- создание и редактирование карты;
- установка точки старта;
- установка точки финиша;
- добавление и удаление препятствий;
- поиск пути алгоритмом **BFS**;
- поиск пути алгоритмом **DFS**;
- поиск пути алгоритмом **A\***;
- пошаговая визуализация поиска;
- отображение посещённых вершин;
- отображение очереди, стека и приоритетной очереди;
- сравнение длины найденного пути;
- импорт карты из файла;
- экспорт результатов поиска в текстовый файл;
- отображение статистики работы алгоритмов.

---

## Используемые алгоритмы

### BFS (Поиск в ширину)

Использует очередь (Queue). Гарантированно находит кратчайший путь при одинаковой стоимости переходов.

### DFS (Поиск в глубину)

Использует стек (Stack). Выполняет поиск в глубину и не гарантирует нахождение кратчайшего пути.

### A*

Использует приоритетную очередь и эвристику Манхэттенского расстояния. Обычно находит кратчайший путь быстрее, чем BFS.

Убедитесь, что установлен Python 3.10 или выше.

Запуск:

```bash
python main.py
```

## Работа с программой

1. Нарисуйте препятствия.
2. Установите стартовую точку.
3. Установите конечную точку.
4. Выберите алгоритм поиска (BFS, DFS или A*).
5. Дождитесь завершения поиска.
6. При необходимости сохраните карту или экспортируйте результат.

## Docker

Приложение работает внутри контейнера с виртуальным дисплеем, а интерфейс доступен через веб-браузер. Не требуется установка X-сервера на хосте.

### 1. Сборка образа
docker build -t pathfinder .

2. Запуск контейнера
docker run -p 6080:6080 pathfinder
Порт 6080 пробрасывается на хост. Если он занят, используйте другой, например -p 6081:6080.

После запуска не закрывайте терминал – контейнер должен работать.

3. Открытие интерфейса
Перейдите в браузере по адресу:
http://localhost:6080/vnc.html

Вы увидите рабочий стол с запущенной программой. Управляйте мышью и клавиатурой.

4. Остановка контейнера
Нажмите Ctrl+C в терминале, где запущен контейнер, или

В другом терминале выполните:

docker ps                  # найти ID или имя контейнера
docker stop <ID или имя>
5. Автоматическая очистка (рекомендуется)
Чтобы контейнер удалялся после остановки, добавьте флаг --rm:

docker run --rm -p 6080:6080 pathfinder
6. Запуск в фоновом режиме
docker run -d -p 6080:6080 pathfinder
Остановка: docker stop <ID>.

7. Если порт 6080 занят
Используйте другой порт, например:

docker run -p 6081:6080 pathfinder
Затем открывайте http://localhost:6081/vnc.html.
